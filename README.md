# 🤖 Warehouse Package Sorting Robot
Autonomous package sorting robot using ROS Noetic & Gazebo Simulation

## 📌 Overview
This robot detects Red, Green, and Blue packages and autonomously sorts 
them into colour-coded bins using a pick-and-place controller.

## 🛠️ Tech Stack
- ROS Noetic (Ubuntu 20.04)
- Gazebo Physics Simulator
- Python, XML (URDF, Launch files)
- LIDAR + RGB Camera

## 🚀 How to Run
1. Clone the repo into your catkin workspace src folder
2. Run: catkin_make
3. source devel/setup.bash
4. roslaunch colour_sorting_robot gazebo.launch
